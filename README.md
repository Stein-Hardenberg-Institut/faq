## Nested Accordion

### Markdown

```shell
<details>
<summary>Produkte</summary>

<blockquote> 
<details>
<summary>Wer nutzt MODULO/LIMO bereits?</summary>
      
    Eine Auswahl unserer Kunden finden Sie hier [unten](https://shi-institut.de/referenzen/).
</details>

</blockquote>
</details>
```
